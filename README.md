## Gameplay

<!-- Add a GIF -->

- Watch the path appear.
- Memorize it.
- The path disappears.
- Reach the chest without stepping on the wrong tile.
- Complete all 9 levels.
- Earn Gold, Silver, or Bronze medals based on completion time.
