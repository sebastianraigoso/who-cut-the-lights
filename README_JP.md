# 消えた道 / Who Cut The Lights

日本語

## プロジェクト概要

『Who Cut The Lights』は、Scratchで開発した記憶力パズルゲームです。

プレイヤーは各レベルの開始時に表示される道順を覚えなければなりません。道が消えた後は、記憶だけを頼りに正しいルートを進み、宝箱を目指します。

このプロジェクトは、ゲームアーキテクチャ、イベント駆動プログラミング、状態管理（State Management）、デバッグ、およびソフトウェアドキュメント作成の学習を目的として開発されました。

## ゲームの特徴
全9レベル
ハート（ライフ）システム
タイマーとメダル評価システム
ゲームオーバーとリトライ機能
スキップ可能なイントロ
レベル進行システム

## ゲームプレイ
表示された道を覚える
道が消える
記憶を頼りに進む
間違ったタイルを踏まないように宝箱を目指す
全9レベルをクリアする
クリアタイムに応じて Gold / Silver / Bronze メダルを獲得する

## 技術的な課題
タイマー同期と表示システム
ライフ（ハート）管理
ゲーム状態の遷移管理
イントロスキップ機能
レベル進行ロジック
Scratchにおけるイベント駆動型システムのデバッグ
